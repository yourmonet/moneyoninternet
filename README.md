# Money on Internet (MONET)
Ini adalah duplikat project dari repository pertama MONET. Pada repository ini, terdapat perbedaan dengan repository sebelumnya yaitu di sini menggunakan MySQL sebagai dabatase, sementara repository pertama menggunakan Supabase sebagai database.

Selain itu, untuk kedepannya seperti update fitur dan integrasi payment gateway akan terupdate di repository ini. Artinya repository sebelumya tidak akan dilanjutkan lagi untuk pengembangan (discontinued). Atau tetap dilanjutkan tapi menggunakan sistem pembayaran manual (tidak menggunakan payment gateway).
